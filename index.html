Data Visualization

My name is Sabahat Iqbal. I often work with government-owned data from emerging markets, such as Pakistan. I decided to start compiling some of the data and present it with interactive data visualizations. Making government data more accessible allows journalists, researchers, and voters to engage more deeply with it.

Beyond the usual front-end stack (HTML, CSS, JS), I use d3.js a lot and ReactJS a little.
